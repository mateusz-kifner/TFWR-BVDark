# TFWR-BVDark
This is theme for game TheFarmerWasRepaced, loosely based on vscode dark theme.

[![](Screenshot_20260413_194923_scale.png)](Screenshot_20260413_194923.png)


### Instalation
1. Open Theme dir of TheFarmerWasRepaced game.
2. Copy file `BVDark.tfwrTheme` from this repo into your Theme dir.
3. In game go to settings > global > theme and select BV Dark

 - Windows: %userprofile%AppDataLocalLowTheFarmerWasReplacedTheFarmerWasReplacedThemes
 - Mac: /Users/%userprofile%/Library/ApplicationSupport/com.TheFarmerWasReplaced.TheFarmerWasReplaced/Themes
